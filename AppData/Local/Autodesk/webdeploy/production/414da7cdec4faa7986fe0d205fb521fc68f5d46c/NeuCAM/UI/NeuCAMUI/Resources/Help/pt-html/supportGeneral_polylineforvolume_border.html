Permite especificar em quais tipos de borda o suporte de polilinha é gerado.

Quando definida como Borda livre, a polilinha é gerada em bordas que não atendem a uma parede. Quando definida como Borda à parede, a polilinha é gerada em bordas que encontram uma parede. Quando definida como Tudo, a polilinha é gerada em todas as bordas.
