Este é o diâmetro de extrusão do bocal. Ele determina a largura da trilha que o cálculo do percurso usa. Este não é o diâmetro do filamento que está definido nas Configurações de impressão. Dica: Para compensar a extrusão excessiva ou a extrusão insuficiente, use Multiplicador de largura de extrusão, também nas Configurações de impressão.
