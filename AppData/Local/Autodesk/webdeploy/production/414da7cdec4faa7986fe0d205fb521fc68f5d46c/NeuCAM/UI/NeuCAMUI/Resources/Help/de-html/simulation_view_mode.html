Steuert, wie Sie die Simulation sehen.

Maschinen-Basis - Behält die Ansicht bei, sodass die Maschinen-Basis auf dem Bildschirm stationär angezeigt wird, während die Rotations- und linearen Achsen, das Werkzeug und das Modell relativ dazu bewegt werden.
Modell - Behält die Ansicht bei, sodass das Modell auf dem Bildschirm stationär angezeigt wird, während sich das Werkzeug relativ dazu bewegt.
Werkzeug - Behält die Ansicht bei, sodass das Werkzeug auf dem Bildschirm stationär angezeigt wird, während sich das Modell relativ dazu bewegt. Verwenden Sie diese Option beim Vergrößern, um die Bewegung des Werkzeugs genauer anzuzeigen. Auch nützlich für die Visualisierung von Gewindeschneidoperationen oder Drehoperationen.
