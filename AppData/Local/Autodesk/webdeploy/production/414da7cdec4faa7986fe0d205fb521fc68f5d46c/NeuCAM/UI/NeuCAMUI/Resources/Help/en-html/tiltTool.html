Tilts the tool to avoid collisions between the shaft or holder and the part.

Specify Shaft & Holder clearances (in the Tool tab of this dialog) and select an option from the Avoidance Mode list to specify the direction in which the tool should tilt when avoiding a collision.

Disabled
Collision between holder and part

Enabled
Collision avoided

Warning: Collision avoidance checks only for collisions between the shaft or holder and the part. Use toolpath simulation to check for collisions against the stock.
